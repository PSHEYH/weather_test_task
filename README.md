# 🌦 Weather App

A Flutter application that allows users to search for cities and view weather forecasts.


# Getting Started

Follow the steps below to run the project locally.

# Requirements

Make sure you have the following installed:

- Flutter SDK: 3.29.1
- Android Studio** or VS Code
-  Xcode
-  Java 17 (recommended)


1. Firstly, you need to install dependencies.

```bash
flutter pub get
```
2. Next step is generate freezed models with command: 

```bash
flutter pub run build_runner build --delete-conflicting-outputs
```
3. After that you can run app on Android or IOS

### Android / iOS:

```bash
flutter run
```
